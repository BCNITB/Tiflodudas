Importar agenda de contacto de iPhone a BlindShell.

Para exportar contactos desde iPhone a Blindshell seguir el siguiente procedimiento:

  1. En el iPhone, descargamos la aplicación MyContacts BackUp (se abre en una ventana nueva).
  2. Tras ejecutarla y aceptar los permisos, realizamos la copia de seguridad de los contactos. Nos informará del número que logra localizar.
  3. Generar un archivo con extensión *.VCF, se ofrece la opción de enviarlo por correo sin salir de la aplicación.
  4. Una vez tenemos esste fichero, conectamos BlindShell al ordenador mediante USB.
  5. Copiamos el archivo en la raíz de la memoria interna o tarjeta SD del BlindShell.
  6. En el BlindShell, vamos a: Contactos\Importar contactos desde carpeta”.
  7. Automáticamente detectará el número de contactos totales y nos consultará si deseamos importarlos.

Cambiar el idioma del teléfono.

Seguir el siguiente procedimiento:

  1. Pulsando ***#### (como si se marca un número de teléfono).
  2. Pulsar el botón de descolgar/llamar (Tecla verde) para acceder al Menú de Ajustes Avanzados (settings).
  3. Bajar con el cursor del teclado del teléfono hasta Avanzado Idioma (Advanced Languaje).
  4. Pulsar en el centro de cursor para acceder al menú de cambio de idioma, donde se mostrarán los diferentes idiomas.
  5. Desplazarnos hacia abajo hasta Español España (Spanish Spain).
  6. Pulsar en el centro del cursor.

El equipo se apagará y cuando vuelva a encenderse estará en el idioma que se haya seleccionado.

¿Que significa el mensaje código MMI no valido?

El mensaje Código USSD en ejecución (o código MMI) aparece cuando se marca un código desde el terminal del que se recibe respuesta desde la red para efectuar alguna operación tal como comprobar o activar desvíos . Suelen ser códigos que van precedidos de símbolos, como * o #, que les hace diferenciarse de una llamada. Si el código es incorrecto o el servicio no está disponible, el mensaje que devuelve la red es: Se ha producido un problema de conexión o el código MMI no es válido.

Solucionar el error de Código MMI.

En ocasiones, cuando marcamos alguno de los contactos de nuestro terminal, introducimos de forma accidental de manera previa, o entre medias de los números, algún símbolo, y nos aparece también el mensaje: Se ha producido un problema de conexión o el código MMI no es válido, en cuyo caso debemos verificar que hemos introducido la numeración correcta, sin nada por el medio.

El error también puede ser debido a un fallo de la tarjeta SIM que puede estar dañada o tener un mal contacto, para confirmar que este no sea el origen del problema, habría que insertarla en un terminal distinto y realizar la prueba desde otro para así descartar que la tarjeta esté mal. Si la incidencia se presenta con ambos terminales, para solucionarlo bastaría con realizar un duplicado de la tarjeta SIM.

Nota: Hay que tener en cuenta que para los teléfonos compatibles con el servicio de Voz 4G no se garantiza la gestión de servicios cuando el usuario tramita los mismos con Comandos USSD ( **21*123# …… etc ).
