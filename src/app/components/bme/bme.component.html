Configuración Instalación Todo
Cuando se da ualgno de estos supuestos:
  • Los protocolos de seguridad utilizados por HTTPS no están activos.
  • Bien configurados.
  • La entidad de confianza que certifica la autenticidad de la web de ONCE no está reconocida.
  • Las librerías del ClickOnce incluidas no permiten hacer descargas seguras.

La instalación de aplicaciones como eBrai, Edico , MIO o Braitico generan un error que indica que no se puede descargar un archivo necesario para instalar.

Seguir el siguiente procedimiento:

  1. Reiniciar el PC, ya que después de tratar de realizar una instalación que da error en el equipo se queda un proceso en memoria que bloquea algunos de los siguientes pasos.
  2. Configurar GlobalSign, que es la entidad que certifica las páginas web de ONCE, como entidad de certificación raíz de confianza. Para ello, hay que descargar el Certificado de GlobalSign y abrir el fichero descargado para proceder a instalarlo.
    1. En la ventana con las propiedades del certificado pulsar Instalar certificado...
    2. Siguiente
    3. Activar la casilla Colocar todos los certificados en el siguiente almacén.
    4. Se abilitará el botón Examinar...
    5. Pulsarel botón habilitado.
    6. En la lista que aparece se debe seleccionar Entidades de certificación raíz de confianza.
    7. Pulsar Aceptar.
    8. Siguiente.
    9. Finalizar.
    10. Aparecerá un aviso de que se va a instalar un certificado de GlobalSign que deberá ser confirmado pulsando Sí.
    11. Cuando el proceso se realice aparecerá un mensaje indicando que La importación se completó correctamente.
  3. En Windows 7, cuando las actualizaciones automáticas no se han llevado a cabo de forma completa, no están activas las descargas TLS 1.2. Deben habilitarse ya que este es el protocolo necesario para las descargas HTTPS que utiliza la web del CTI. Existe una aplicación de Microsoft que realiza este proceso de forma automática, y que puede descargarse en Easy Fix.
    1. Ejecutar el fichero.
    2. Aceptar la licencia.
    3. Siguiente”.
    4. Conceder los permisos que se soliciten.
    5. Aparecerá el mensaje: La operación se ha realizado correctamente y el botón Cerrar.
    6. Pulsar sobre el botón.
  4. Principalmente en equipos con Windows 7, aunque podría ocurrir también en Windows 8 y Windows 10, puede ser necesario instalar el Microsoft .NET Framework 4.6.2 para que ClickOnce funcione con TLS 1.2 descargando el instalador desde el la página de Microsoft.
    1. Ejecutar el fichero descargado.
    2. Conceder los permisos que se soliciten.
    3. Aceptar la licencia marcando la casilla que lo indica.
    4. Instalar.
    5. Finalizar.
  5. Cambiar la política de seguridad relacionada con el protocolo de seguridad SSL que impide el acceso a la web de la ONCE. Para ello seguir los siguientes pasos:
    1. Iniciar sesión con un usuario que tenga permisos de administrador.
    2. Abrir el Editor de directivas locales, pulsando Windows+R, tecleando gpedit.msc y pulsando Aceptar.
    3. En el árbol de directivas que aparece navegar hasta; Directiva Equipo local > Configuración del equipo > Plantillas administrativas > Red > Opciones de configuración SSL.
    4. En la lista de directivas seleccionar la que se llama Orden de la curva ECC y modificarla. Su estado debe quedar en No configurada.
    5. Pulsar Aceptar.
  6. Una vez modificada la directiva, se debe reiniciar el PC para que los cambios surtan efecto.
