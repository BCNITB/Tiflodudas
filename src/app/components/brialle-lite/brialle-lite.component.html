Instalación Todo

C: Al instalar el programa Ebrai y poner la activación se me abre en modo comentarios y no permite seleccionar impresora, configurar página o imprimir. Se ha desinstalado el programa y vuelto a instalar sin cambios.

R: Es necesario instalar el Visual C++ Redistributable de Microsoft Visual Studio. Para ello sigue estos pasos:

  1. Descargar el programa de instalación de Visual Studio Redistributable. Asegurate de bajar la versión adecuada para su sistema operativo, X86 para Windows de 32 bits, X64 para Windows de 64 bits.
  2. Cierra EBrai (si está ejecutándose) e instala Visual C++ Runtime.
  3. Vuelve a lanzar EBrai.

Las opciones de impresora ya deberían estar disponibles.
