C: Cambiar la configuración del teclado de una línea focus a una mano.

R: Utilzar los siguientes comandos para permutar entre una configuración y otra:

  • Shift izquierdo + 1-2-3-7.
  • Shift derecho + 4-5-6-8.

Si se configura mediante menñus, seguir el siguiente procedimiento para activar o desactivar el modo a una mano:

  1. Pulsar la tecla menú.
  2. Para acceder a la opción, pulsar Punto 8.
  3. Para cambiar entre activado y desactivado usar el Balancín izquierdo.
  4. Para confirmar la opción seleccionada, pulsar Punto 8.
