Cambiar la configuración del teclado de una línea focus a una mano.

Utilzar los siguientes comandos para permutar entre una configuración y otra:

  • Shift izquierdo + 1-2-3-7.
  • Shift derecho + 4-5-6-8.

Si se configura mediante menús, seguir el siguiente procedimiento para activar o desactivar el modo a una mano:

  1. Pulsar la tecla menú.
  2. Para acceder a la opción, pulsar Punto 8.
  3. Para cambiar entre activado y desactivado usar el Balancín izquierdo.
  4. Para confirmar la opción seleccionada, pulsar Punto 8.
Conexión línea braille con TalkBack.

Para añadir una línea braille a Talkback debemos tener instalada BrailleBack (dependiendo de la versión de Android no está incluido por defecto) y realizar los siguientes pasos:

  1. Activar la conexión mediante Bluetooth.
  2. Vicular la línea braille.
  3. Ir a:
    1. Ajustes.
    2. Accesibilidad.
    3. BrailleBack.
  4. Comprobar que la línea braille está enlazada, si no lo est´´a pulsar sobre el desplegable y seleccionarla.

Nota

:
  • Actualmente BrailleBack está incluido en la Suite de Accesibilidad de Android.
  • Para que la línea funcione debe estar activado TalkBack.

La línea Focus está bloqueada.

Para desbloquear la línea braille:

  1. Desconectar la línea braille de todos los dispositivos.
  2. Pulsar, con la línea encendida, Modo derecho y dos sensores durante 5 segundos.
Cuando se pulsa un sensor de la línea braille para posicionarse en un lugar determinado de la misma, el cursor sube 3 líneas.

Las siguientes opciones deben estar activadas:

  • Teclas de la línea braille desplazan el cursor activo.
  • La línea braille sigue el cursor activo.

Nota: Estas configuraciones se encuentran en el gestor de configuración > Braille > General.
