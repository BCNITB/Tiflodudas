Los drivers no se actualizan correctamente y la impresora entra en un bucle.

Mantener pulsado le tecla CHS durante 10 segundos.

Conocer la IP de la impresora.

Existen diferentes opciones:

  • En la impresora Index-D V5:
    • Para imprimir la IP pulsar: HELP, 1
    • Para que la impresora diga la IP pulsar: HELP, 10
  • En Windows realizar lo siguiente:
    1. Panel de Control.
    2. Impresoras.
    3. Propiedades de la impresora.
    4. Puertos.
Al realizar una impresión con Index Everest DV-5 desde Ebrai se realiza una doble paginación.

Se debe desactivar la paginación de la Index Everest o de la aplicación Ebrai.
