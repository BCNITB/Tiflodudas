Compatibilidad Navegación

C: Cuando se abre directamente un archivo con extensión *.opd, se ejecuta Omnipage pero se abre un archivo en blanco.

R: Hay que hacer el proceso a la inversa:

  1. ejecutar Omnipage.
  2. Ir a Archivo.
  3. Abrir.
  4. Buscar el documento.

De esta manera se vé el archivo en pantalla.

C: Si tienes varias ventanas en la pantalla (miniaturas, editor de texto, imágenes, etc….) si que dando F6 va cambiando de ventana o de área como lo hace con el Power Point, pero desactivando todas y dejando sólo el editor de texto, no se sabe, ni visualmente, dónde va el foco.

R: Cada vez que se termine de reconocer un texto o se haya abierto el archivo, aunque ya se hubiera seleccionado la opción, volver a repetir los pasos de acceder a la barra de menú de OmniPage > Ver > Editor de textos > Texto sin formato.
