Inicio Inicio

La presente aplicación busca ser una referencia rápida para resolver algunos d los problemas más comunes que se han documentado durante la instalación o la ejecución de programas para facilitar la accesibilidad a diferentes aparatos:

  • Smartphones.
  • Ordenadores.
  • BlinsShells.
  • Reproductores Daisy.
  • Lineas braille.
  • Etcétera.

Si se quiere, se puede descargar fichero apk para instalar la aplicación en un dispositivo Android de forma local.

Mediante el menú lateral podemos acceder a los diferentes componentes de la página:

  • Consultar incidencias.
  • Buscador de incidencias.
  • Enviar/Resolver incidencia.
  • Enviar incidencias con su solición.
