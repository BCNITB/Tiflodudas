¿Para qué sirve?

La presente aplicación web busca ser una referencia rápida para resolver algunos d los problemas más comunes que se han documentado durante la instalación o la ejecución de programas para facilitar la accesibilidad a diferentes aparatos:

  • Smartphones.
  • Ordenadores.
  • BlinsShells.
  • Reproductores Daisy.
  • Lineas braille.
  • Etcétera.

Descarga de la app

Si se quiere, se puede descargar fichero apk para instalar la aplicación en un dispositivo Android de forma local.

¿Consultar o Buscar en la base de datos?

Tanto el a parte inferior de esta página como en el menú lateral existen dos opciones: Consultar la Base de Datos y Buscar en la Base de Datos. El comportamiento de dichas opciones es diferente.

  • Consultar la Base de Datos: En este apartado se muestra la Base de datos entera, pero de forma local (si se ha descargado la aplicación) o mediante código HTML las consultas solucionadas.
  • Buscar en la Base de Datos: En esta apartado se muestran todas las consultas, con o sin solución desde una base de datos, y sólo estarán disposibles si hay acceso a internet.

La idea actualizar la base de datos de forma bisemanal o mensual (dependiendo de las nuevas aportaciones), y actualizar la aplicación cada seis meses (dependiendo de las aportaciones), para que las soluciones esténdisposnibles en la aplicación de forma local y sin necesidad de acceder a internet.

Diferentes apartados

Mediante el menú lateral podemos acceder a los diferentes componentes de la página:

  • Consultar incidencias.
  • Buscador de incidencias.
  • Enviar/Resolver incidencia.
  • Enviar incidencias con su solución.
